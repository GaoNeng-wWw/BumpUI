# Center

## 基本用法

```tsx
import React from 'react';
import { Center } from '@qwqui/core';

const App = () => {
  return (
    <Center>
      <span>Your content here</span>
    </Center>
  );
};

export default App;
```


## Props

| 名称 | 类型 | 介绍 |
|---|---|---|
| children | `React.ReactNode` | 要居中的内容。 |
| inline | `boolen` | 如果为`true`，那么`display`属性则为`inline-flex`,否则`display`属性则为`false` |
